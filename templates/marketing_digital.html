SEO: Posicionamiento en buscadores

El posicionamiento en buscadores denominado Search Optimization Engines conocido por su sigla SEO se compone por un conjunto de acciones que debe realizar en una página web para aparecer mejor posicionado en los resultados de búsqueda naturales. Este conjunto de acciones incluyen los factores técnicos, el desarrollo de contenido y popularidad.
Para lograr esto es importante comprender el funcionamiento de un buscador, para esto los invito a ver el video de Matt Cutts ( Google) que explica este proceso:



Principales Etiqutas SEO

A continuación se muestra la estructura html típica de una página web y se muestra en color aquellas etiquetas o atributos que guardan mayor relación con el posicionamiento en buscadores.

<!DOCTYPE html>
<html>
<head>El contenido de la etiqueta head no se muestran al usuario, contiene información útil para el navegador y los buscadores
<title>Título de la página</title>
<meta name= "Description" content="Se describe el contenido de la página"/>
<meta name= "Keywords" content="Incluye las palabras claves separadas por coma"/>

<link href="css/estilos.css" rel="stylesheet" type="text/css">

<script src="scripts/jquery.js" type="text/javascript"></script>

</head>

<body>Todo el contenido incluido dentro del body se muestra al usuario

<h1> Encabezado de primer nivel </h1>

<h2> Encabezado de segundo nivel </h2>

<a href="url a donde apunta el link" target="_blank" <title=”Esto es un tool.tip”><Texto ancla</a>

<img src="imagenes/mapa.PNG" <alt="La etiqueta alt describe la imagen para que pueda ser leida por los buscadors">title="Título de la imagen">

<p> La etiqueta p se usa para indicar un parrafo </p>

</body>
</html>

Para ver el código fuente podemos usar los atajos de teclado:
Pulsa Ctrl+U (Windows) o ⌘-Opción-U (Mac).
